# rollup-jquery-boilerplate

Simple jquery + es6 boilerplate using [Rollup](https://github.com/rollup/rollup) and [Bubl√©](https://gitlab.com/Rich-Harris/buble). 
There's no gulp, webpack, grunt or anything.

## Install

``` 
git clone https://github.com/fri13th/rollup-jquery-boilerplate
cd rollup-jquery-boilerplate
npm install
```

## Build

``` 
npm run build
``` 

## Watch

``` 
npm run watch
``` 

## Uglify

``` 
npm run production
``` 

## Config

You can set distribution file name in build.js.
``` 
const appName = 'app';
```
